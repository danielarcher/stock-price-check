## Stock Price Check

This is a Laravel web application using Blade, AlpineJS and Tailwind css.

- [Laravel](https://laravel.com).
- [AlpineJS](https://alpinejs.dev/).
- [Tailwind Css](https://tailwindcss.com/).

## Set Up

```
$ composer install
$ docker-compose up -d
$ get 
```
